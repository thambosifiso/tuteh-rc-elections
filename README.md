# TUTEH RC Elections Voting System

A secure, transparent voting system for TUTEH RC Elections.

## Features
- Unique key numbers for each student
- Secure login with Student ID + Key
- Verified candidate ballot
- One student = one vote
- Real-time vote tallying
- Audit logs for transparency

## Project Structure
- `frontend/` → HTML, CSS, JS for login, ballot, confirmation
- `backend/` → Node.js + Express server with routes
- `database/` → SQL schema for students, candidates, votes, audit logs
